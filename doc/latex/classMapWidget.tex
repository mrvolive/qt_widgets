\doxysubsection{Detailed Description}
Widget personnalisé pour l\textquotesingle{}affichage de tuiles cartographiques. 

Cette classe gère le téléchargement et l\textquotesingle{}affichage d\textquotesingle{}une carte composée de tuiles cartographiques Open\+Street\+Map.
