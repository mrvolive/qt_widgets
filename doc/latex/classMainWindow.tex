\doxysubsection{Detailed Description}
Classe principale de l\textquotesingle{}interface utilisateur. 

Cette classe gère la fenêtre principale de l\textquotesingle{}application, comprenant les menus, les widgets et les layouts. Elle affiche une liste de villes et une carte du monde.
